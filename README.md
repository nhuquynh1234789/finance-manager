# Finance Manager

A full-stack personal finance manager project.

## Folders
- backend: Java backend code
- frontend: HTML/CSS/JS UI
- sql: MySQL schema

## Features
- Add income and expense transactions
- View summary (profit/loss)
- Extendable for reports, charts, login

## How to run
1. Start backend server (Java Spring Boot or similar)
2. Open `index.html` in browser
3. Connect to MySQL database
